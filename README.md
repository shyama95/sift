# SIFT (Scale Invariant Feature Transform)
## Abstract
An implementation of SIFT (Scale-invariant feature transform) from scratch in python v3
## Dependencies
- python v3
- python libraries : opencv (to read/ save images), numpy, matplotlib, scipy (v1.1.0)

~~~~

~~~~

~~~~

~~~~
